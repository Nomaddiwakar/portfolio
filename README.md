# Diwakar Kumar - Software Engineer Portfolio

Personal portfolio website built with modern web technologies showcasing projects, engineering skills, experience, and certifications.

## 🚀 Live Demo
- **Vercel Production**: [https://engineer-digi-vault.vercel.app](https://engineer-digi-vault.vercel.app)
- **GitHub Repository**: [https://github.com/Nomaddiwakar/portfolio](https://github.com/Nomaddiwakar/portfolio)

## 🛠️ Tech Stack
- **Framework & Library**: React 18, Vite, TypeScript
- **Styling & UI**: Tailwind CSS, shadcn/ui, Framer Motion
- **3D & Visuals**: Three.js, @react-three/fiber, @react-three/drei
- **Icons & Components**: Lucide React, Radix UI

## 💻 Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Nomaddiwakar/portfolio.git
   cd engineer-digi-vault
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start local development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
